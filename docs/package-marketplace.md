# Package Installer & Marketplace

Configure system requirements and share profiles with the marketplace.

---

## Package Installer

The Package Installer synchronizes system packages across different environments. Define your required packages in `config.toml` (or using the `config.toml` file generated by `reshell`):

```toml
packages = [
    "git",
    "fzf",
    "bat",
    "ripgrep",
    "fd-find",
    "tmux",
    "lazygit"
]
```

### Elevated Execution (`sudo`)

System package installation requires administrative privileges on Linux (e.g. running `apt-get install` or `pacman -S`). 

To handle this cleanly:
- When running `reshell install` or selecting install in the TUI, reshell detects if your package manager requires elevation.
- If it does, you are prompted securely for your password.
- This password is piped to `sudo -S` standard input, streaming installation feedback directly to the viewport log screen.
- The manager checks if a package is already installed (by scanning your `PATH` or querying local package manifests) to skip redundant operations.

## Marketplace Packs

Marketplace packages let teams and communities share configuration setups. 

### Importing Packages

Run:
```bash
reshell install github.com/aaryansinhaa/reshell-java
```

This performs the following actions:
1. Clones the remote Git repository into a temporary directory.
2. Checks for `reshell.toml` in the repository root.
3. Imports the defined aliases, snippets, and environment variables into your configuration.
4. Appends any required package names to your global configuration list.
5. Copies all function files inside the repository's `functions/` folder to your local `~/.config/reshell/functions/` directory.
6. Copies all categorized scripts inside the repository's `scripts/` folder to your local `~/.config/reshell/scripts/` directory.

### Manifest Schema (`reshell.toml`)

Here is an example `reshell.toml` manifest file for a marketplace package:

```toml
[package]
name = "reshell-java"
description = "Java terminal configurations for developers"

[[aliases]]
name = "jrun"
value = "java -jar"
description = "Run a JAR file"
shell = "all"
enabled = true

[[variables]]
name = "JAVA_HOME"
value = "/usr/lib/jvm/java-17-openjdk-amd64"
description = "Java Home path"
enabled = true

[[snippets]]
name = "mvn-clean-install"
code = "mvn clean install -DskipTests"
description = "Maven build without running tests"
tags = ["maven", "java", "build"]
language = "bash"
shell = "all"

[config]
packages = ["openjdk-17-jdk", "maven", "gradle"]
```
